JavaScript Variables: VAR==> Global Scope

javaScript variables are containers for storing data values. In this example, x, y, and z, are variables: x stores the value 5 y stores the value 6 z stores the value 11

JavaScript Identifiers

All JavaScript variables must be identified with unique names. These unique names are called identifiers. Identifiers can be short names (like x and y) or more descriptive names (age, sum, totalVolume).

The Assignment Operator

In JavaScript, the equal sign (=) is an "assignment" operator, not an "equal to" operator. In JavaScript, however, it makes perfect sense: it assigns the value of x + 5 to x. (It calculates the value of x + 5 and puts the result into x. The value of x is incremented by 5.)

JavaScript Data Types

JavaScript variables can hold numbers like 100 and text values like "John Doe". In programming, text values are called text strings. JavaScript can handle many types of data, but for now, just think of numbers and strings. Strings are written "inside" double or single quotes. Numbers are written without quotes.

JavaScript Const

Variables defined with const behave like let variables, except they cannot be reassigned JavaScript const variables must be assigned a value when they are declared:

Incorrect: const PI; PI = 3.14159265359;

Correct: const PI = 3.14159265359;

You can change the properties of a constant object:

Correct

// You can create a const object: const car = {type:"Fiat", model:"500", color:"white"}; // You can change a property: car.color = "red"; // You can add a property: car.owner = "Johnson";

Incorredt

const car = {type:"Fiat", model:"500", color:"white"}; car = {type:"Volvo", model:"EX60", color:"red"}; // ERROR

You can change the elements of a constant array:

Correct

// You can create a constant array: const cars = ["Saab", "Volvo", "BMW"]; // You can change an element: cars[0] = "Toyota"; // You can add an element: cars.push("Audi");

Incorrect

const cars = ["Saab", "Volvo", "BMW"]; cars = ["Toyota", "Volvo", "Audi"]; // ERROR
